Hungry Hiragana!

Welcome to the website

Hungry Hiragana is an interactive website designed to teach you Hiragana using mnemonics and a multiple-choice quiz. Start your Hiragana journey by clicking the learning button above and then come back to do the test when you are ready. There is also lots more information about Hiragana whenever you feel like taking a break from studying!

Why learn Hiragana?

Being able to read and write Hiragana is the first step to learning Japanese which is an extremely interesting, useful and unique language. With this skill you will be able to understand Japanese pronounciation and soon move on to learning Katakana and Kanji!
